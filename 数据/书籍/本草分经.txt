
序


考《汉书·艺文志》，分医经与经方而二之。而隋书经籍志则统以医方。乃读其论，则均有味乎谚所谓有疾弗治。适得中医云云盖诚见夫医或失宜，热以益热，寒以增寒，转伤于内，与其授权于庸医，固不若听命于造物，以不治治。然瞑眩瘳疾载在书，医师率属，十全为上，亦着于周官，安得废医，夫亦曰审之而已。《神农本草》，《汉志》未登，肇见之《隋书》，嗣而《唐书》新旧，而《宋史》悉录之，即国朝《四库全书》，采医家言至九十七种，言本草者亦十余种。在神农初，药分三品，共三百六十种。迨雷公桐君，增其族类，广其主治，而书益着，而医经资之达其用，而经方之名，因以立。所以由汉而隋，史氏遂统以医方称也。且夫医书之存于今者，亦仅矣。夫《针经》、《素问》，古所谓《黄帝内经》也，次若《难乃与针灸家所共援，以察荣卫部位脏腑脉法、经络 穴。是史论医经，所谓原人、血脉阴阳表里，以起百病之本，死生之分，慎度所施。而后之人，有因以畅其说者，如《金匮要略》、《伤寒论》，及《病源候》，及三因五运六气等论，皆医经之遗也。余则各本草，乃史论经方，所谓本草木之寒温，辨五苦六辛，致水火之齐，因气感之宜，以通蔽解结。而后之人，有因以善制造者，如《千金》、《百一》、《和剂局方》，及《全生》、《卫生奇疾》等方，皆经方之类也。即至意见有偏，门户各立，有若河间易州金华，或主泻火，主滋阴，莫不各先研经图后，依于本草而讲修治。然则经图及本草，其医之始事欤。今诸医经，既难率购，而《四库》所录诸本草，坊间亦少刊布。即所市有《本草备要》从新各编，又绝不及经图，徒掇拾破碎，未易会通而酌其宜。山阴友人姚建霞茂才来省，其兄清如于靖安署，示予以本草分经一编。编中略采录诸经脉图，具列诸药总目，再析列通行分行，不循经络各类中，各以补和攻散寒热，自为其类。不务详核，而尚简当，甚便检阅。熟审之，依论为方，当不至益热增寒，而疾可瘳，十全可几。惜板毁于兵，无从复购。因为正其次序，复付手民公诸世，增名为《本草分经》审治。审之者谁？阅此者事也。编着者谁？实建霞同堂曾王父讳澜号 云广文先生也，而名以和尚，则以晚年寡发，故又冠以维摩。维摩诘？盖深入法门，明了众生，能断众生病者，想诸经图其究之精矣。乃若本草，实医之始事也，是编之着。
不过如药王药上王子，尝世界草木金石，悉知苦酢咸淡甘辛等味，并诸和合，是冷是热有毒无毒。如《 严经》所云，尔维摩学佛，服行无量功德，当不仅以是编。狎视之，维摩有知，其将亦点首乎。黄梅梅雨田。

原叙


本草之作，肇自神农，厥后代有传书，至纲目而大备。然卷帙浩繁，艰于记诵。于是肤浅者，流率以药性赋为宗旨，挂一漏百，贻害无穷。迨备要从新诸书行于世，而后本草之功用复着，顾其体例，则仍以草木虫鱼，分门而比类。读者但识其性味主治，而于所入之经络，每多忽之，此所以有诛伐无过之讥，而难于收针芥相投之效也。吾友山阴姚君，名澜字 云，申韩高手也。由明经需次广文，余筮仕之江，即延之宾馆，论交垂三十年间。遇微病君为治之，则信手拈来，药止数味，而效如桴鼓。询以岐黄，曰：“吾非知医，但知某药入某经耳。”庚子之春，余摄篆越郡，得姚君所辑《本草分经》抄本。公余之暇，披览一过。盖以经络为纲，以药品为目，俾阅者豁然于某味为某经之药，不致乱投杂进，其有裨于医术不浅哉。爰急为付梓，以广其传。姚吾素善病，中年须发尽脱，因自号维摩和尚，今已逾花甲，而精力不衰，殆即按经服药之明效欤。

原例（次序略经更定）

简介
一、编内先列内景经络诸图，以资考镜，且使病患自觉何处为患，即可知为何经之病，宜用何经一、十二经次序：始于手太阴肺，次手阳明大肠，次足阳明胃，次足太阴脾，次手少阴心，次手太阳小肠，次足太阳膀胱，次足少阴肾，次手厥阴心包，次手少阳三焦，次足少阳胆，终于足厥阴肝。是编不依次开列者，亦因便于翻阅故也。
一、是编以经络为纲，药品为目。势不能于一经之内汇草木虫鱼之全，故总目全载药品，又于每一、凡一药而兼入数经者，均于总目每味之下注明。至其性味功用，则止于第一经之一味内一、药有不循经络者，另列杂品一门。凡总目不下注某经，或通行者，皆杂品也。
凡一经汇一经之药，从其同也。而其功用则各不同，故又分列补和攻散寒热六者，使之亦从其同。庶令阅者，依类取用，较为便捷。
一、药品多有一物数名者，若分载各味之下，则散而杂稽，兹将同名诸品，汇列一卷，以便一、药性有畏恶反忌，读本草者，不可不知。然古方多有兼用者，若泥于其性而不知变通，转多窒滞。是当广阅古方，以求其义，不必存胶柱之见也。故编内不备载畏恶反忌之文。
一、是编所载药名，及其字体，概从时俗。如薏苡仁作米仁，恶实作牛蒡子，又 作姜，石不妨从俗，而在初学则便于查阅也。

通行经络



<span style='font-weight:bold;'>（补）人参</span>
甘温微苦，大补肺中元气，其性主气，凡脏腑之有气者皆能补之。生阴血亦泻虚火，凡服参不投者服山楂可解，一补气一破气也。按：老山真参近时绝少，惟行条参其性味与人参虽同而力极薄，出关东，不论大小，但须全糙白皮为上，半糙者次之，若皮色微黄虽糙难辨红熟者多伪，不可用。修条力甚薄，而其性横行手臂，指臂无力者服之有效。参须与修条相同，其力尤薄。参芦能涌吐痰涎，虚者用之以代瓜蒂，然亦能补气，未见其尽吐也。


<span style='font-weight:bold;'>高丽参</span>
气味略似人参，而性较温。初服似有力，数日后便不觉矣。野者不可得，种者愈大愈佳。东洋参 野者皮白，状类西洋参而色香味无异人参，性则微凉，近皆种者，形似人参而性温，闻种时皆用硫黄故也。若以之代党参，较为轻清，非可代人参也。


<span style='font-weight:bold;'>黄精</span>
甘平，补气血而润安五脏，益脾胃润心肺，填精髓助筋骨，除风湿。


<span style='font-weight:bold;'>大枣</span>
甘温，补中益气，滋脾土润心肺，调营卫通九窍，助十二经，和百药，脾病患宜食之。加入补剂与姜并行，能发脾胃升腾之气。风疾痰疾俱非所宜。红枣功用相仿，而力稍逊。
南枣不入药。生枣甘辛，多食生寒热。


<span style='font-weight:bold;'>面</span>
甘温，补虚养气，助五脏浓肠胃，北产陈麦良，新麦热，南产壅气助湿热。


<span style='font-weight:bold;'>鹿肉</span>
甘温，补中强五脏，通脉益气力。


<span style='font-weight:bold;'>羊肉</span>
甘热，属火，补虚劳益气力，开胃壮阳道，能发痼疾及疮。羊 结成羊腹中者，治反胃。羊角明目杀虫。生羊血治血晕，解一切毒。


<span style='font-weight:bold;'>鳝鱼</span>
甘大温，补五脏，去风湿，能走经络。


<span style='font-weight:bold;'>淡菜</span>
甘咸温，补五脏益阳事，治虚劳消 气。


<span style='font-weight:bold;'>人乳</span>
甘咸，纯阴无定性，润五脏补血液，清烦热理噎膈，利肠。有孕之乳为忌乳，最有毒。


<span style='font-weight:bold;'>人气</span>
治下元虚冷，日令童男女以时隔衣进气脐中，甚良。或身体骨节痹痛，令人更互呵熨，久久经络通透。


<span style='font-weight:bold;'>紫河车</span>
甘咸温，大补气血而补阴之功尤为极重，治一切虚劳损极，大有奇效，且根气所钟必达元海。病由膀胱虚者用之尤宜。清水洗至净白，用铅壶隔汤煮极烂，连汁入药。或煮略熟文火焙干用，有胎毒者伤人，须以银器试之。


<span style='font-weight:bold;'>（和）甘草</span>
味甘，通行十二经，解百药毒，生用气平，补脾胃、泻心火、而生肺金；炙用气温，补三焦元气而散表寒。入和剂则补益，入汗剂则解肌，入凉剂则泻热，入峻剂则缓急，入润剂则养血，能协和诸药，使之不争。头涌吐，消上部肿毒。梢达茎中。


<span style='font-weight:bold;'>香附</span>
辛香微苦微甘，通行十二经八脉气分，调一切，气能引血药至气分而生血，解六郁利三焦消积调经，乃治标之品，损气耗血。


<span style='font-weight:bold;'>连翘</span>
见心和。


<span style='font-weight:bold;'>合欢皮</span>
甘平，和血补阴，安五脏和心志，盖心脾调和则五脏自安矣。


<span style='font-weight:bold;'>芜荑</span>
辛苦温，散满KT 湿，化食杀虫，祛五脏、皮肤、肢节风湿，能疗鳖瘕虫痛。


<span style='font-weight:bold;'>海桐皮</span>
苦平，入血分，祛风去湿杀虫，能行经络达病所，治牙虫癣疥。


<span style='font-weight:bold;'>乳香</span>
苦温，辛香善窜，入心，通行十二经，调气活血，去风舒筋，托里护心，香彻疮孔，能使毒气外出，消肿止痛生肌。


<span style='font-weight:bold;'>没药</span>
苦平，入十二经，散结气通瘀血，消肿定痛生肌。


<span style='font-weight:bold;'>竹沥</span>
甘苦寒滑，清痰降火，行经络、四肢、皮里、膜外之痰，凡痰因风热燥火者宜之，姜汁为使。虚者与参同用，使人参固其经，竹沥通其络，则甘寒气味相得益彰。


<span style='font-weight:bold;'>荆沥</span>
甘平，开经络除风热，化痰行气血，为去风化痰之妙药，用牡荆俗名黄荆，烧取沥。


<span style='font-weight:bold;'>广皮</span>
见肺和。


<span style='font-weight:bold;'>枳子</span>
甘平，止渴润五脏，解酒毒。按：葛根、葛花解酒毒而发散不如枳 。


<span style='font-weight:bold;'>菠菜</span>
甘温而滑，利五脏通血脉，开胸膈，下气调中，止渴润燥，根尤良。


<span style='font-weight:bold;'>荠菜</span>
甘温，利五脏，益肝和中。根益胃明目，同叶烧灰治痢。


<span style='font-weight:bold;'>白豆</span>
甘平，补五脏，暖肠胃，调中，助十二经脉，肾病宜食之。豆叶利五脏、下气。豆腐甘咸寒，清热散血，和脾胃消胀满，下大肠浊气。


<span style='font-weight:bold;'>酒</span>
大热有毒，用为向导可以通行一身之表，引药至极高之分，和血行气，逐秽暖水脏，最能乱血动火致湿热诸病，醇而无灰、陈者良。按：锻石能解酒酸，造酒家多用之，而有灰之酒伤人。烧酒散寒破结，损人尤甚。


<span style='font-weight:bold;'>灵砂</span>
甘温，养神志安魂魄，通血脉调和五脏，治上盛下虚痰涎壅盛、吐逆、冷痛，杀精鬼，小儿惊吐服之最效，为镇坠神丹也。硫黄合水银炼成。


<span style='font-weight:bold;'>百沸汤</span>
助阳气，行经络，半沸者饮之伤元气作胀。


<span style='font-weight:bold;'>鹈鹕油</span>
咸温滑，透经络，治聋痹、痈肿诸病，不入汤丸。


<span style='font-weight:bold;'>蜂蜜</span>
甘滑，生性凉清，热熟性温，补中润燥解毒，调营卫通三焦，安五脏通便秘，止诸痛和百药，与甘草同功，滑肠。同葱食害人，食蜜饱后食 令人暴亡。黄蜡甘淡微温性涩，止痛生肌，续绝伤止泻痢。


<span style='font-weight:bold;'>（攻）大戟</span>
苦辛寒，专泻脏腑水湿，逐血发汗，消痈通二便闭，泻火逐痰，其汁青绿亦能泻肝。阴寒善走，大损真气。紫色者上，白者伤人，须去骨用，中其毒者惟菖蒲能解之。


<span style='font-weight:bold;'>甘遂</span>
苦寒，泻肾经及隧道水湿，直达水气所结之处，以攻决为用，治大腹肿满、痞积、痰迷，去水极神，损真极速，面煨用。


<span style='font-weight:bold;'>商陆</span>
苦寒，沉阴下行，与大戟甘遂同功，疗水肿、胀满、蛊毒、恶疮。


<span style='font-weight:bold;'>芫花</span>
苦温，疗五脏水饮、痰癖、治瘴疟，毒性至紧，虚者忌之，醋煮用。根疗疥。


<span style='font-weight:bold;'>防己</span>
大辛苦寒，入膀胱，去火邪，能行十二经，通腠理利九窍，泻下焦血分湿热，疗风行水，降气下痰。性险而健，惟湿热壅遏及脚气病凡下焦湿热致二阴不通者用此治之。有二种，汉防己治水用，木防己治风用。


<span style='font-weight:bold;'>鹤虱</span>
苦平，杀五脏虫，治蛔痛。


<span style='font-weight:bold;'>巴豆</span>
辛大热大毒，峻下开窍宣滞，去脏腑沉寒积滞，治喉痹急症。生用急治，炒黑缓治，去油名巴豆霜，大黄、黄连、凉水、黑豆、绿豆汁能解其毒。


<span style='font-weight:bold;'>苏木</span>
甘咸辛平，入三阴血分，行血去瘀，因宣表里之风。


<span style='font-weight:bold;'>枳实</span>
苦酸微寒，破气行痰，消痞止喘，利胸膈宽肠胃。


<span style='font-weight:bold;'>枳壳</span>
性味功用与枳实同，惟实则力猛而治下，其泻痰有冲墙倒壁之功，壳则力缓而治上，能损胸中至高之气，为异耳。


<span style='font-weight:bold;'>角刺</span>
辛温，搜风杀虫，通窍溃痈，其锋锐直达病所。


<span style='font-weight:bold;'>槟榔</span>
苦辛温，能坠诸药下行，攻坚破胀消食，行痰下水散邪，杀虫醒酒，泻胸中至高之气至于下极，凡气虚下陷者宜慎用。


<span style='font-weight:bold;'>轻粉</span>
辛冷燥毒，劫痰涎，消积杀虫，善入经络，不可轻服，今人用治杨梅毒疮，能劫邪从牙龈出，然毒入经络筋骨，血液耗亡多成痼疾，惟土茯苓能解其留毒。粉霜略同。


<span style='font-weight:bold;'>虻虫</span>
苦寒有毒，攻血，遍行经络，色青入肝，极能堕胎。


<span style='font-weight:bold;'>穿山甲</span>
咸寒，性猛善窜，入肝胃，功专行散，能出入阴阳，贯穿经络，入营分以破结邪直达病所。通经下乳，消肿溃痈，止痛排脓，和伤发痘，为风疟疮科要药。

甘寒，理气化痰益肠胃。


<span style='font-weight:bold;'>黄豆</span>

<span style='font-weight:bold;'>豌豆</span>
甘平属土，治吐逆泄痢腹胀。


<span style='font-weight:bold;'>黎豆</span>
甘微苦温，益气温中。


<span style='font-weight:bold;'>荞麦</span>
甘寒，降气利肠，治肠胃沉积。


<span style='font-weight:bold;'>野麦</span>
甘平，滑肠可救荒。


<span style='font-weight:bold;'>麦</span>
甘微寒，补中除热。


<span style='font-weight:bold;'>米醋</span>

<span style='font-weight:bold;'>自然铜</span>

<span style='font-weight:bold;'>古文钱</span>
辛平，治目中障瘀横产五淋，亦可煮汁。


<span style='font-weight:bold;'>密陀僧</span>
辛平，镇惊劫痰，止血消积，杀虫疗肿毒灭瘢 ，治疮痔，出银坑，今以倾银炉底代之。


<span style='font-weight:bold;'>白矾</span>
酸咸寒，性涩而收，燥湿涌涎化痰，除风止血，通二便杀虫，除痼热在骨髓，生用解毒，多服损心肺伤骨。


<span style='font-weight:bold;'>绿矾</span>
酸凉涩收，能燥湿消积化痰，利小便解毒杀虫，主治略同白矾。


<span style='font-weight:bold;'>煤炭</span>
甘辛温，治气血痛及痰痫疮伤，中其毒者以冷水解之。


<span style='font-weight:bold;'>无名异</span>
甘咸，和血生肌治疮伤。


<span style='font-weight:bold;'>石燕</span>
甘凉，利窍行湿热，治淋带目障。


<span style='font-weight:bold;'>石蟹</span>
咸寒，明目解金石药毒。


<span style='font-weight:bold;'>立春节雨水</span>
甘平，宜煎发散及补中益气药，又立春清明二节贮水亦名神水，浸造诸风脾胃虚损丹丸，久留不坏。


<span style='font-weight:bold;'>小满芒种白露三节雨水</span>
皆有毒，造药酿酒易坏，饮之生脾胃疾。


<span style='font-weight:bold;'>梅雨水</span>
洗疮疥灭瘢痕，入酱易熟。


<span style='font-weight:bold;'>端午午时雨水</span>
宜造疟痢疮疡虫蛊诸丹丸。


<span style='font-weight:bold;'>神水</span>
端午午时有雨，急伐竹竿中必有水名为神水，甘寒清热化痰定惊安神，治心腹积聚及虫病。


<span style='font-weight:bold;'>寒露冬至大寒小寒四节雨水</span>
宜浸造滋补药及痰火积聚虫毒丹丸。


<span style='font-weight:bold;'>腊日雨水</span>
与寒露冬至大寒小寒雨水同。


<span style='font-weight:bold;'>液雨水</span>
立冬后十日为入液，至小雪为出液，能杀百虫，宜煎杀虫消积之药。


<span style='font-weight:bold;'>霜</span>
甘寒，解酒热敷痱疮。


<span style='font-weight:bold;'>腊雪水</span>
甘寒，治时行瘟疫伤寒火 ，抹痱，春雪有虫不用。


<span style='font-weight:bold;'>冰</span>
甘寒，治伤寒阳毒昏迷，解烧酒毒。


<span style='font-weight:bold;'>潦水</span>
甘平，宜煎调脾胃去湿热之药，降注雨水为潦，淫雨亦为潦。


<span style='font-weight:bold;'>半天河</span>
竹篱头及空树穴中水也，甘微寒，治鬼邪蛊毒，洗疮。


<span style='font-weight:bold;'>东流水</span>
性顺疾速，通膈下关，荡涤邪秽。


<span style='font-weight:bold;'>逆流水</span>
性逆而倒上，宣吐痰饮。


<span style='font-weight:bold;'>井水</span>
甘凉，清热助阴，平旦新汲者佳。


<span style='font-weight:bold;'>醴泉</span>
甘平，治鬼气邪秽及心腹痼疾。


<span style='font-weight:bold;'>乳穴水</span>
近乳穴处流出之泉也，甘温，久服肥健能食体润不老。


<span style='font-weight:bold;'>玉井水</span>
有玉山内之泉水也，甘平，久服体润毛发不白。


<span style='font-weight:bold;'>温泉</span>
辛热微毒，患癣疥风癞杨梅疮者饱食，入池久浴取汗。


<span style='font-weight:bold;'>阿井水</span>
甘咸平，性趋下，清而且重，治瘀浊及逆上之痰，下膈止吐。


<span style='font-weight:bold;'>泉水</span>
出山岩间者是也，甘平，治霍乱烦闷呕吐，山有毒草恶石者不可用。


<span style='font-weight:bold;'>海水</span>
咸微温，有小毒，浴风癣吐，下宿食胪胀。


<span style='font-weight:bold;'>地浆</span>
甘寒，阴中之阴，治热毒中 ，解诸毒。


<span style='font-weight:bold;'>生熟汤</span>
调和阴阳治霍乱吐泻。


<span style='font-weight:bold;'>水</span>
即作黄 菜水也，酸咸，吐痰饮宿食。


<span style='font-weight:bold;'>桑柴火</span>
能助药力凡一切补药诸膏宜此火煎之。


<span style='font-weight:bold;'>栎炭火</span>
力紧宜 炼金石药。


<span style='font-weight:bold;'>炭火</span>
力慢宜烹煎炙焙百药丸散。


<span style='font-weight:bold;'>芦火</span>
其力不强，不损药力，宜煎一切滋补药。


<span style='font-weight:bold;'>竹火</span>
与芦火同。


<span style='font-weight:bold;'>灯火</span>
去风解毒通经，惟麻油苏子油燃者可用，余皆损目亦不治病。


<span style='font-weight:bold;'>灯花</span>
止血生肉，敷金疮，治小儿夜啼。


<span style='font-weight:bold;'>黄土</span>
甘平，治泻痢热毒兼解诸毒。


<span style='font-weight:bold;'>东壁土</span>
甘温，治瘟疫泄痢疗疮癣，东壁得，初日烘炙少火之气壮，取真火所照之土，引真火发生之气以补土胜湿，或用南壁土，取离火所照之气，用西壁土取西方收敛之气，皆借气以补脾胃也。


<span style='font-weight:bold;'>伏龙肝</span>
多年灶心黄土对釜穴下者是也，辛温功专去湿，亦能调中止血消肿催生。


<span style='font-weight:bold;'>釜脐墨</span>
辛温，止血消积，治血病蛊毒伤寒阳毒，涂金疮。


<span style='font-weight:bold;'>百草霜</span>
灶突上烟煤也，辛温，止血消积，治血病及伤寒阳毒口舌诸疮。


<span style='font-weight:bold;'>梁上尘</span>
辛苦微寒，止血消积，治噎膈中恶小儿软疮，烧令烟尽筛取末用。


<span style='font-weight:bold;'>墨</span>
辛温，止血生肌涂痈肿。


<span style='font-weight:bold;'>鹅</span>
甘温有毒，发疮动风，火熏者尤甚。血治反胃噎膈。蛋甘温补中益气。


<span style='font-weight:bold;'>鸽</span>
咸平，解药毒，治疮癣及人马久患疥，白色者入药。蛋解疮毒痘毒。屎名左盘龙，治痞块阴毒人马疮疥，野鸽者尤良。


<span style='font-weight:bold;'>白丁香</span>
苦温微毒，消积，治疝瘕、积胀、疮疽、咽噤、齿目诸病。


<span style='font-weight:bold;'>象皮</span>
治金疮，长肌肉，外用。


<span style='font-weight:bold;'>猫胞</span>
甘酸温，治反胃吐食。肉治劳疰鼠 蛊毒。


<span style='font-weight:bold;'>猪獾</span>
甘酸平，长肌肉，治劳热水胀。


<span style='font-weight:bold;'>狗獾</span>
甘酸平，补中益气，小儿疳瘦者宜食之。


<span style='font-weight:bold;'>獭肝</span>
甘咸温，杀虫治传尸痨，疗鱼骨哽。肉甘咸寒，治骨蒸血热便秘，消阳气。


<span style='font-weight:bold;'>鼠矢</span>
甘微寒，治伤寒劳复阴易腹痛，两头尖者为雄鼠矢。胆明目治聋。肉治儿疳鼠 。


<span style='font-weight:bold;'>白蜡</span>
甘温，止血生肌，补虚续筋骨。


<span style='font-weight:bold;'>原蚕砂</span>
二蚕矢也，辛甘温，去风燥湿，治风湿诸病，炒熨患处亦良。


<span style='font-weight:bold;'>壁钱</span>
即 子窠，治喉痹虫牙痛及疮口不敛。


<span style='font-weight:bold;'>绯帛</span>
治恶疮肿毒，作膏用又敷小儿脐未落时肿痛。五色帛治堕马及一切筋骨损，拭盗汗。


<span style='font-weight:bold;'>鲚鱼</span>
甘温无毒，发疥助火动痰。


<span style='font-weight:bold;'>鲈鱼</span>
甘平有小毒，和肠胃，治水气发疮肿，安胎。


<span style='font-weight:bold;'>鳜鱼</span>
甘平无毒，益气力补虚，去腹内恶血小虫，治肠风泻血。


<span style='font-weight:bold;'>鱼</span>
疗水肿利小便，治口眼 斜，非佳品也，勿多食。


<span style='font-weight:bold;'>黄颡鱼</span>
甘平微毒，发疮疥消水肿，利小便，反荆芥，害人。


<span style='font-weight:bold;'>河豚鱼</span>
甘温有大毒，味虽美，修治失法常杀人。


<span style='font-weight:bold;'>比目鱼</span>
甘平无毒，补虚益气力，多食动气。


<span style='font-weight:bold;'>金鱼</span>
甘咸平，治久痢。


<span style='font-weight:bold;'>海蛇</span>
咸平，治妇人积血小儿风疾。


<span style='font-weight:bold;'>瓦楞子</span>
甘咸平，消痰破血癖。


<span style='font-weight:bold;'>人骨</span>
治骨病 疮，能接骨，取焚叶者，胎骨有毒服之伤生。


<span style='font-weight:bold;'>脐带</span>
止疟，解胎毒，敷脐疮。


<span style='font-weight:bold;'>指甲</span>
性平，治难产去目中翳障，取孕妇指上者。


<span style='font-weight:bold;'>口津唾</span>
甘咸平，辟邪明目消肿毒。


<span style='font-weight:bold;'>月水</span>
咸热而毒，解毒箭，治女劳复。月经衣熨金疮血涌，烧服治虎野狼伤及箭镞入腹。


<span style='font-weight:bold;'>裤裆</span>
洗汁治女劳，复烧灰治阴阳易，男病用女，女病用男，取近阴处者。


<span style='font-weight:bold;'>（攻）荛花</span>
苦辛微寒，行水破积聚症瘕，荡涤肠胃饮食痰饮，治伤寒温疟。


<span style='font-weight:bold;'>败蒲</span>
草名也，破血治霍乱恶疮跌扑瘀血。


<span style='font-weight:bold;'>藜芦</span>
辛寒至苦，治蛊毒喉痹，杀虫，入口即吐，风痫症用之服后吐不止者饮葱汤即止，与酒同用杀人。


<span style='font-weight:bold;'>茹</span>
辛寒，破血排脓，蚀恶肉杀疥虫。


<span style='font-weight:bold;'>常山</span>
辛苦寒性猛烈，能引吐行水祛痰饮截疟，与甘草同用或生用，多用则吐，若酒浸炒透但用钱许，未见其尽吐也。


<span style='font-weight:bold;'>马鞭草</span>
苦微寒，破血消胀杀虫，治症瘕疽毒。


<span style='font-weight:bold;'>使君子</span>
甘温，杀虫治小儿疳积，多食伤脾食，后饮热茶作泻。


<span style='font-weight:bold;'>天名精</span>
辛甘寒，破血吐痰，泻热解毒。根名杜牛膝，功用相同，洗痔良。


<span style='font-weight:bold;'>刘寄奴</span>
苦温，破血下胀除症，止金疮血，多服令人吐利。


<span style='font-weight:bold;'>续随子</span>
辛温，行水破血，解毒利肠，长于利水而攻击猛鸷，虚者勿服，研去油用。


<span style='font-weight:bold;'>凤仙子</span>
微苦温，软坚透骨通窍，治产难骨哽，消积块，最能损齿与玉簪根同。花甘温而滑，活血消积治蛇伤。根叶苦甘辛，散血软坚，治杖扑肿痛鸡鱼骨哽及误吞铜钱。


<span style='font-weight:bold;'>蓖麻子</span>
辛甘热，能开通诸窍经络，出有形滞物，利水气拔毒，外用屡效，内服宜慎食，蓖麻一生不得食炒豆，犯之胀死。


<span style='font-weight:bold;'>大枫子</span>
辛热，治疮疥杀虫劫毒。


<span style='font-weight:bold;'>杉木</span>

<span style='font-weight:bold;'>柞木</span>

<span style='font-weight:bold;'>肥皂</span>
辛温，泻热毒除风湿，去垢腻疗疡毒。


<span style='font-weight:bold;'>干漆</span>
辛温毒烈，功专行血杀虫，破年深积滞瘀血，炒至烟尽用，得蟹则成水。


<span style='font-weight:bold;'>八角金盘</span>

<span style='font-weight:bold;'>莱菔子</span>
辛温破气，生用能吐风痰散风寒，炒熟治喘嗽下痢消食止痛，其治痰有冲墙倒壁之功。


<span style='font-weight:bold;'>银朱</span>
辛温燥烈，破积滞劫痰涎，兼疗疮疥杀虫虱。


<span style='font-weight:bold;'>水银</span>
辛寒阴毒性滑，重直入肉，功专杀虫解五金毒，堕胎绝孕，同枣肉人唾研则碎。


<span style='font-weight:bold;'>锻石</span>
辛温毒烈，燥湿散血生肌，灭瘢疵杀疮虫，风化者良。古矿灰火毒已出，治顽疮敛疮口。


<span style='font-weight:bold;'>砒石</span>
辛苦酸，大热大毒，燥胸膈之痰，可作吐药，杀虫，出信州锡之苗也，炼者名砒霜尤烈。


<span style='font-weight:bold;'>石</span>
辛大热而毒，攻寒积，性气与砒石相近。


<span style='font-weight:bold;'>砂</span>
咸苦辛热，消食破瘀，热毒之性能烂五金、化人心为血。


<span style='font-weight:bold;'>硝石</span>
辛苦微咸，大热毒烈，破积散坚。


<span style='font-weight:bold;'>碱</span>
辛苦涩温，消食磨积去垢，除痰点痣疣发面。


<span style='font-weight:bold;'>虫</span>
咸寒有毒，搜剔血积，接折伤治木舌通乳。


<span style='font-weight:bold;'>蝼蛄</span>
咸寒，性急有毒，行水，腰以前甚涩，能止二便，腰以后甚利，能通二便。


<span style='font-weight:bold;'>水蛭</span>
咸苦平有毒，破血治恶血积聚及丹毒，可染须。


<span style='font-weight:bold;'>斑蝥</span>
辛寒毒，专走下窍，逐败物治石淋瘰 ，溃肉堕胎下 犬毒，外用蚀死肌、敷疮疥。


<span style='font-weight:bold;'>蜂房</span>
甘平有毒，杀虫，治痈疽惊痫牙痛。


<span style='font-weight:bold;'>鼠妇</span>
酸温微寒，治气癃月闭血瘕寒热，利水道堕胎。


<span style='font-weight:bold;'>蛴螬</span>
咸微寒有毒，治血瘀、痹气胁下坚满、破折、金疮，下乳疗目疾，以背反行者真。


<span style='font-weight:bold;'>蜣螂</span>

<span style='font-weight:bold;'>牙齿</span>

<span style='font-weight:bold;'>木贼草</span>

附余


人参、丹参、沙参、苦参、元参、紫参、细辛、芍药皆与藜芦相反，大戟、芫花、甘遂、海藻皆与甘草相反，半夏、栝蒌、贝母、白蔹、白芨皆与乌头相反，石决明反云母，硫黄反芒硝，乌头反犀角，水银反砒霜，巴豆反牵牛，丁香反郁金，芒硝反三棱，肉桂反石脂，藜芦反酒黄，颡鱼反荆芥，葱韭与蜜俱相反，醋与蛤肉相反，牛乳与酸物、生鱼相反，人参畏五灵脂，野狼毒畏密陀僧， 皮与桔梗、麦冬相恶。之类大抵相反则彼我交仇，不宜合用，非比相畏相恶尚可制伏也。
按：人参畏五灵脂，恶皂荚，反藜芦，而东垣交 丸用人参、皂荚，是恶而不恶也。古方疗月闭四物汤，加人参、五灵脂，是畏而不畏也。又治痰在胸膈，人参、藜芦同用。而取其涌越，是激其怒性也。又甘草反大戟、芫花、甘遂、海藻，而胡洽治痰癖，十枣汤以芫花、大戟、甘遂煎枣汤入药末加甘草。东垣治结核，甘草与海藻同用。丹溪治劳瘵，甘草与芫花同用。又黄 畏防风，而玉屏风散防风与黄 同用。牛黄恶龙骨，而龙骨得牛黄反良。非洞奥达权不能知也。
凡药根升而梢降。
五味主用，苦泄甘缓，酸收咸软，淡渗泄，辛散，辛甘发散为阳，酸苦涌泄为阴。
苦药平升，微寒平亦升，甘辛药平降，甘寒泻火，苦寒泻湿热，苦甘寒泻血热。
味之薄者阴中之阳，酸苦咸平是也；味之浓者阴中之阴，酸苦咸寒是也；气之浓者阳中之阳，辛甘温热是也；气之薄者阳中之阴，辛甘淡平凉寒是也。
